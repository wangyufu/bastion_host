堡垒机
==
作者:王宇夫    

## 程序介绍:     
    使用python3+、django1.11，Windows或Linux环境使用；
	利用paramiko源码和shellinabox实现了web的主机管理；
	利用paramiko和多进程实现了批量命令和批量文件的功能。
